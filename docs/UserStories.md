# AstraNotes User Stories

As a user, I want to create notes so that I can save information.

As a user, I want to edit notes so that I can update information.

As a user, I want to delete notes so that I can remove unwanted information.

As a user, I want to search notes so that I can quickly find information.

As a user, I want to organize notes with tags so that related notes can be grouped together.

As a user, I want Markdown support so that I can format my notes.

As a user, I want encrypted storage so that my information remains private.
