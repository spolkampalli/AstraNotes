# AstraNotes Security Notes

## Implemented Security Features

* Note content is encrypted before being stored.
* Encrypted note content is decrypted when displayed.
* Encryption keys are stored locally.
* Sensitive files are excluded from GitHub using .gitignore.

## Privacy Considerations

* User notes remain on the local machine.
* No cloud services are required.
* No user data is transmitted externally.

## Future Improvements

* Password-protected encryption keys
* User authentication
* Secure backup and recovery
