# AstraNotes Governance Memo

## Responsible AI Use

AI was used to assist with requirements generation, documentation, UML planning, and implementation guidance.

All AI-generated content was reviewed, refined, and validated before inclusion in the final project.

## Privacy Considerations

Users may store sensitive information in notes. User data should remain local and should not be shared with AI systems.

## Security Considerations

The project implements encrypted note storage and local data persistence.

## Human Oversight

Human judgment was used to:

* Select project scope
* Refine requirements
* Validate generated code
* Review documentation
* Verify testing results
