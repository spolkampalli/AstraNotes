# AstraNotes Product Requirements Document (PRD)

## Product Overview

AstraNotes is a lightweight note-taking application that allows users to create, edit, search, organize, and securely store notes locally.

## Problem Statement

Many users need a simple note-taking solution without relying on cloud services or complex software. AstraNotes provides local storage, organization features, and privacy-focused design.

## Target Users

* Students
* Professionals
* Personal note-taking users

## Core Features

* Create notes
* Edit notes
* Delete notes
* Search notes
* Tag notes
* Markdown formatting
* Local storage using SQLite
* Encrypted note storage

## Out of Scope

* Multi-user collaboration
* Mobile applications
* Cloud synchronization
* Plugin marketplace

## Success Criteria

Users can create, edit, search, organize, and securely store notes through an intuitive interface.
